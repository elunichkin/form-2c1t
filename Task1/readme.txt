Задание 17.

Для разбора регулярного выражения, заданного в обратной польской записи, будем использовать стек пар целых чисел. Первый элемент пары - максимальная длина слова, состоящего целиком из букв x (-INF, если такого слова нет; 0 для пустого слова). Второй элемент пары - максимальная длина суффикса, состоящего из букв x. Каждый раз (с приходом очередного символа) будем проверять выражение на корректность.
'a', 'b', 'c':
Если текущий символ равен x, то положим в стек (1,1), иначе (-INF,0).
'+':
Извлечём из стека 2 верхних элемента и положим в стек пару, содержащую максимумы первых и вторых элементов извлечённых пар соответственно.
'*':
Если максимальное слово из букв x непустое (first > 0), то положим в стек (INF,INF) (слово стало бесконечным). Иначе положим туда (0,second), так как максимальное слово не появилось, а суффикс (собственный) не увеличился
'.':
При конкатенации будем выполнять несколько проверок (все if'ы достаточно очевидны из кода) и положим в стек получившуюся пару.
